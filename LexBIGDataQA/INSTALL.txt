INSTALL
-------
1. Choose $QA_HOME
2. Unzip dist_lexbigQa*.zip into $QA_HOME
3. Configure $QA_HOME/etc/db.prop for your environment
4. Read through documentation in $QA_HOME/doc

Dependencies
-----------------------
Subsystem Dependencies
  $JAVA_HOME

Network Dependencies
  Mysql connection - port 3306

Required Perl (5.8+) Libraries
  DBI
  DBD::mysql
   
UPDATE
------
Same as install.
